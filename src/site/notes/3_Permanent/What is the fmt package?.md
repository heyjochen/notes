#type/permanent 

Used for formatting input and output similar to C's printf and scanf.

Print a Line:
```go
func main() {
    fmt.Println("Hello, Go!")
}
```

Read Input:
```go
func main() {
    var inputString string
    fmt.Print("Enter your name: ")
    fmt.Scan(&inputString)
    fmt.Println("Hello, " + inputString + "!")
}
```
